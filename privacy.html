Privacy Policy

Last updated: April 2025

MonAgentTiktok is a personal desktop application that generates and publishes videos to TikTok.

Data Collection

This application does not collect, store, or share any personal data. It only interacts with the authenticated user's TikTok account via the official TikTok API.

Third Party Services

This app uses the TikTok Content Posting API solely to publish videos on behalf of the authenticated user.

Contact

For any questions, contact us via GitHub.
